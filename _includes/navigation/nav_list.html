The original markup mismatched
  • nesting (the `{% if %}` for sublinks did not close its parent
  • before `{% endif %}`), producing invalid HTML that confused assistive tech. 2. Used inline `style="padding-left: 0;"` instead of utility classes. 3. Did not respect the documented page.sidebar.nav schema. Migration: Replace `{% include navigation/nav_list.html nav="foo" %}` with `{% include navigation/nav-tree.html nav_key="foo" %}`. The block below is now a thin wrapper that delegates to nav-tree.html so any straggling call sites continue to function during the deprecation window. It also emits a build-time warning when JEKYLL_LOG_LEVEL=debug. =================================================================== {%- endcomment -%} {%- if include.nav -%} {% include navigation/nav-tree.html nav=include.nav %} {%- endif -%}
