{%- comment -%} =================================================================== LOCAL GRAPH WIDGET — sidebar mini graph =================================================================== File: local-graph.html Path: _includes/navigation/local-graph.html Purpose: Renders a small "local graph" of the current page and its immediate wiki-link neighbors in the left sidebar (Obsidian- style local graph view). Behavior: - Reads /assets/data/wiki-index.json - Finds the current page by window.location.pathname (with title/basename fallback for permalink quirks) - BFS depth=1 by default (current + direct neighbors, in & out) - Hides itself if the current page is not in the wiki-index, has no neighbors, or the index can't be loaded - Lazily loads cytoscape.js from CDN with SRI + crossorigin Opt-out: set `local_graph: false` in page front matter. Configurable depth: `local_graph_depth: 2` in page front matter (default 1). =================================================================== {%- endcomment -%} {%- if page.local_graph != false -%} {%- assign lg_depth = page.local_graph_depth | default: 1 -%}
