{% comment %} Feature: ZER0-008 {% endcomment %} {%- comment -%} Component: sidebar-folders Path: _includes/navigation/sidebar-folders.html Purpose: "collection" sidebar mode — live folder tree of a collection's documents, grouped by sub-folder into collapsible sections. Zero-config: any collection gets a usable sidebar without a curated _data/navigation/*.yml file. Params: collection (string, optional) — collection label to list (default: page.sidebar.collection, then page.collection) sort (string, optional) — "path" | "title" | "date" (default: page.sidebar.sort, then "path") Front matter (page.sidebar hash, all optional): collection — list a different collection than the page's own sort — see above; "date" needs date/lastmod front matter reverse — true to reverse the sort (e.g. newest-first) expand — true to expand every folder group (default: only the group containing the page) Per-document opt-out: set `sidebar_exclude: true` in a doc's front matter to hide it from this tree. Collection metadata (optional, in _config.yml `collections.
