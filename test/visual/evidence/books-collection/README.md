# Evidence — books collection (ZER0-079)

Visual evidence for the `books` collection: the `book.html` landing layout,
the `book-story.html` reading layout, and the chapter-order navigation.
Generated by [`test/visual/books-evidence.mjs`](../../books-evidence.mjs)
(reuses the shared evidence-kit primitives) against the theme's built-in
demo book, `pages/_books/zer0-tales/`.

This is a **new feature**, so the montages are after-only — the "before"
state is a 404 (no books collection existed).

| File | Shows |
|---|---|
| `01-book-landing.png` | Cover hero (portrait book frame, byline, synopsis, Start-reading CTA) + ordered TOC at 1280px and 390px |
| `02-book-story.png` | Reading surface: chapter header, storybook serif with drop cap, full-width illustration plates, The End flourish, at 1280px and 390px |
| `03-story-navigation.png` | Prev/next derive from numeric `chapter:` order — chapter 1 disables Previous, the last chapter disables Next, Contents returns to the book |
| `metrics.json` | Page-overflow sweep for both routes across 320–1440px — **0px everywhere** |

Regression coverage: [`test/visual/features/books.spec.js`](../../features/books.spec.js)
(smoke tier, `@critical`) pins the H1/cover/TOC structure, the bounded serif
reading measure with drop cap, phone-width plate containment, the
navigation states, and the collapsed prompt colophon.
