The only files you need in your repository for GitHub Pages to render a full site through the bamr87/zer0-mistakes remote theme.

The minimum viable configuration:

{% highlight yaml %} # _config.yml title: "My Site" description: "A site rendered by the zer0-mistakes remote theme." url: "" baseurl: "" # Pull layouts/includes/assets from the published theme remote_theme: bamr87/zer0-mistakes # GitHub Pages whitelisted plugins plugins: - jekyll-remote-theme - jekyll-feed - jekyll-sitemap - jekyll-seo-tag # Hide this welcome screen once you're configured site_configured: true {% endhighlight %}
