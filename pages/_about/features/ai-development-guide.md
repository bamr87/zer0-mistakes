#### **VS Code Extensions**

```json
{
  "recommendations": [
    "github.copilot",
    "github.copilot-chat",
    "yzhang.markdown-all-in-one",
    "bradlc.vscode-tailwindcss",
    "ms-vscode.vscode-json",
    "redhat.vscode-yaml"
  ]
}
```

### **Initial Configuration**

#### **Workspace Settings**

Create `.vscode/settings.json`:

```json
{
  "github.copilot.enable": {
    "*": true,
    "yaml": true,
    "markdown": true,
    "liquid": true
  },
  "github.copilot.advanced": {
    "enableSuggestionAutoTrigger": true,
    "contextLength": 4096
  },
  "files.associations": {
    "*.md": "markdown",
    "*.html": "liquid"
  }
}
```
