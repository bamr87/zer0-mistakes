# Mermaid with Native Markdown Syntax

**‚ú® Simple Implementation:** No HTML tags needed! Just use markdown code blocks.

---

## üöÄ Quick Start

### Step 1: Write Markdown Code Blocks

Simply use triple backticks with `mermaid` as the language:

````markdown
```mermaid
graph TD
    A[Start] --> B[End]
```
````

### Step 2: That's It!

No front matter needed, no HTML `<div>` tags, just pure markdown.

---


## üîß Configuration

The plugin is configured in `_config.yml`:

```yaml
plugins:
  - jekyll-mermaid

mermaid:
  src: 'https://cdn.jsdelivr.net/npm/mermaid@10/dist/mermaid.min.js'
```

**That's all the configuration needed!**

## üìù Usage Summary

### In Your Markdown Files

Just use standard markdown code blocks:

````markdown
```mermaid
graph LR
    A --> B --> C
```
````

### No Front Matter Needed

Unlike the previous implementation, you don't need:

```yaml
---
mermaid: true  # ‚Üê NOT NEEDED ANYMORE
---
```

### No HTML Tags Needed

Unlike the previous implementation, you don't need:

```html
<div class="mermaid">  <!-- ‚Üê NOT NEEDED ANYMORE -->
graph LR
    A --> B
</div>
```

## üéì Quick Reference

### Flowchart Directions

```
TD or TB - Top to bottom
BT - Bottom to top
LR - Left to right
RL - Right to left
```

### Node Shapes

```
[Rectangle]
(Rounded)
{Diamond}
((Circle))
>Flag]
```

### Arrow Types

```
-->  Solid arrow
-.-> Dotted arrow
==>  Thick arrow
--   Line without arrow
```
