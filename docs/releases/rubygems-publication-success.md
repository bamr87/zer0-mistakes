### Basic Sitemap Implementation
```yaml
# sitemap.md
---
title: Site Map
layout: sitemap-collection

permalink: /sitemap/
---

Welcome to the comprehensive site map with statistics and navigation tools.
```

### Navigation Integration
```yaml
# _data/navigation/main.yml
- title: Site Map
  url: /sitemap/
  icon: bi-map
  sublinks:
    - title: Statistics
      url: /stats/
    - title: Content Overview
      url: /sitemap/
```

### Collection Display
```yaml

```
